{% trans "General informations" %}

{% blocktrans %}This library is divided in two parts: the real library on the one hand and the sharing system on the other hand. Both part do interact and you probably can not use one without the other one. The ownership system could seem a little bit tricky but once explained, you will find it naturally simple.{% endblocktrans %}

{% trans "The library" %}

{% url "book_list" as book_list_url %} {% blocktrans %}

The library contains books. These books have a title*, a published date, an author. You can also link the book with general themes or with periods of literature history. Each book can be possessed by many people and each people can have several copies of a same book. When you go onto the book list, you can see different options (depending on your permission level). You can edit a book or delete it.

{% endblocktrans %}
{% blocktrans %}When you delete a book, you also delete all the ownerships and the lendings linked to a book, so do it carefully ! If you want to remove it from your library without deleting the book in the database, you can use the remove button. If you have enough privileges, you will be then also allowed to remove it from others libraries.{% endblocktrans %}
{% blocktrans %}

You can also add books to your own library: these books, that belongs to other libraries, have already been added to the database. You do not need to create them again, all you need is to add it your library. You will then have the chance to specify how many copies you have. If you want, you can also add a specific editor, a specific cover or insert some comments. This way, the book is added to your own library and everyone now knows that you have one copy of it.

{% endblocktrans %}
